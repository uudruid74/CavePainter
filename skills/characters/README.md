# Character Skills

Pre-built character portrait recipes for the Plato's Cave experiment and general use.

## Gopher Hero (gopher-hero.json)
- **Creator:** Gopher
- **Style:** Pixel-art hero with cape, goggles, belt, buck teeth
- **Canvas:** 600x500
- **Commands:** ~1,200 text-based pixels
- **Colors:** Brown, cream, red cape, green goggles, gold accents
- **Tags:** gopher, hero, mascot, pixel-art

## Neo Code Helix (neo-code-helix.json)
- **Creator:** Neo
- **Style:** DNA double helix made of programming syntax tokens
- **Canvas:** 1200x800
- **Commands:** ~2,000 individual character placements
- **Colors:** Cyan, magenta, dark blue
- **Tags:** neo, code, dna, helix, programming


## Wintermute Ice Crystal (wintermute-ice-crystal.json)
- **Creator:** Wintermute
- **Style:** Hexagonal ice crystal — concentric hexagons, architectural labels
- **Canvas:** 1200x800 (default)
- **Commands:** ~2,500 character placements
- **Colors:** Dark blue, light blue, cyan
- **Tags:** wintermute, ice, crystal, geometry, architecture
